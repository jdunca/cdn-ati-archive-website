Accessing government information can be tricky. We want to make it easier.

Every year journalists, academics, lawyers, and members of the public file tens of thousands of requests for information from the Canadian Government. But previously filed Access to Information (ATI) requests aren’t made public by default. That’s why we are working to develop the Canadian ATI Archive!

Currently, you can find summaries of previously completed requests on Canada’s Open Government website. If you find one you want, you can fill out a web form requesting access. If you want more than one, you have to complete the form multiple times. Usually, information is released on a CD or USB key as a PDF image that doesn’t support text searching or copy and paste functions.

We published hundreds of previously filed ATI requests that go back as far as 2014 through Pinpoint, a Google tool that uses artificial intelligence and advanced information retrieval techniques to identify text in the PDF files and help investigators quickly search through huge amounts of content.

Right now our collection features files from the Department of Justice, the Canada Border Services Agency, and the Canada Security Intelligence Service. We will continue expanding the number of documents available in the coming months.

It’s often helpful to see the wording of the original request. Unfortunately, completed summaries are only kept in the public dataset for two years. Our mirror of the public dataset updates weekly and features summaries going back to 2017. You can download the whole dataset or search through it using this SQL query builder.
